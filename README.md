# Lista de endereços

## O que é
Um app simples feito em react para cadastrar e listar endereços

## O que dá pra fazer
- Cadastrar um endereço
- Validar os campos do formulário
- Ver a lista de endereços cadastrados

## Tecnologias
- React
- TypeScript
- HTML
- CSS

## Como rodar no vscode
1. Abra o terminal na pasta do projeto
2. instale as dependências: npm install
3. Rode o projeto: npm run dev
4. abra o link que aparecer no terminal, geralmente: "localhost/5173/"
